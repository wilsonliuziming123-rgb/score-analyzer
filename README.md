# Student Score Analyzer

A full-stack JavaScript web app for analyzing student scores.

## Features

- Enter scores separated by commas, Chinese commas, spaces, or new lines
- Validate scores from 0 to 100
- Calculate count, max, min, mean, median, Q1, Q3, IQR, range, standard deviation, passing count, and failing count
- Show animated charts and warning messages
- Save the latest input and result in the browser

## Run Locally

```bash
npm install
npm start
```

Open:

```text
http://localhost:3000
```

## Deploy

Recommended platform: Render.

Use these settings:

```text
Build Command: npm install
Start Command: npm start
```
